

This is an archive of programs used to do
invariant checking for ptograms on lists.

It is part of the CINV distribution, see

http://www.liafa.jussieu.fr/cinv


PREREQUISITES
-------------
- CINV tool version >= 0.2.2
  
- GCC

INSTALLATION
------------
- Install CINV and set the environement variables
	o CINV
	o LD_LIBRARY_PATH
  as mentioned in this distribution.

RUNNING
-------
- Build the binary for each example by

	> make example.exe

- Execute the binary

	> ./example.exe
